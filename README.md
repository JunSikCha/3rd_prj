# 🔑 핵심기능

### 1. 온라인 강의 수강

> 사용자는 학생 회원 혹은 강사 회원으로 가입 후 로그인할 수 있습니다. <br/>
> <br/>학생 회원으로 로그인 시 원하는 강좌를 수강 신청하여 강의를 수강할 수 있고, 테스트 응시 후 수료할 수 있습니다. <br/>
> <br/>강좌마다 강사에게 직접 문의를 남길 수 있고, 웹 사이트에 문의사항이 있을 시 일반 문의를 작성할 수 있습니다.

### 2. 강좌 업로드

> 강사 회원으로 로그인 시 본인의 강좌를 업로드할 수 있습니다. 시험 문제 등을 관리합니다. <br/>
> <br/>학생 회원이 문의한 내용에 대해 답변을 할 수 있고, 강좌 공지사항을 게시할 수 있습니다.

### 3. 웹 사이트 관리

> 관리자로 로그인 시 전반적인 사이트의 관리를 할 수 있습니다. <br/>
> <br/>강사 회원이 업로드한 강좌에 대해 승인하거나 반려할 수 있습니다. <br/>
> <br/>회원 정보를 관리할 수 있습니다. <br/>
> <br/>회원들이 게시한 일반 문의에 대해 답변할 수 있고, 웹 사이트의 공지사항을 게시할 수 있습니다.
